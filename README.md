## コマンド一覧
qrプログラムでは，以下のコマンドでドローンを操作できます．

line traceプログラムでは，終了コマンドと離陸コマンドのみ使用可能です．
- ESC : 終了
- p : ライントレース開始
- t : 離陸
- l : 着陸
- w : 前へ
- s : 後ろへ
- a : 左へ
- d : 右へ
- r : 上へ
- c : 下へ
- e : 右回り
- q : 左回り
